# SpendSat-Bitcoin-Trust-Spending-Infrastructure
A decentralized trust-layer for Bitcoin commerce in Africa. Solving the merchant-spender trust gap using Citrusrate API and Orange Pillar verification. [PM Portfolio Project]
Role: Lead Product Manager

## Project Vision
Enabling real-world Bitcoin spending in emerging markets by solving the "Trust Gap" between merchants and spenders.

##  The Problem
Bitcoin adoption in Africa is high for trading but zero for daily spending because:
1. Merchants fear volatility.
2. Spenders lack trust in informal merchants.
3. No reputation layer exists for circular economies.

## The Solution (My PM Strategy)
I led the team to build a non-custodial marketplace using **Citrusrate.com API** with three core pillars:
- **Merchants:** Simplified listing with fiat-to-sats conversion.
- **Spenders:** Location-based discovery and instant Lightning payments.
- **Orange Pillars:** A decentralized verification layer to vouch for merchant quality.
